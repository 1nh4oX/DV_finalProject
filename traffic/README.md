# 🚗 深圳城市交通与出行模式分析

## 数据集说明

### 数据来源
1. **模拟数据** - 基于真实交通特征生成
2. **高德API数据** - 可选（需要API密钥）

### 数据集包含
1. **hourly_traffic.csv** - 小时交通流量数据
2. **road_congestion.csv** - 路段拥堵数据
3. **metro_ridership.csv** - 地铁客流数据
4. **od_flow.csv** - OD流量数据
5. **travel_mode_share.csv** - 出行方式数据
6. **weather_traffic.csv** - 天气与交通数据
7. **daily_trips.csv** - 日均出行数据
8. **speed_distribution.csv** - 速度分布数据

## 📊 数据说明

### 数据特点
- 时间跨度：30天
- 地理范围：深圳市
- 数据量：10万+ 条记录
- 数据质量：基于真实交通规律生成

### 数据真实性
数据是基于以下真实特征生成的：
- 深圳真实道路名称
- 真实地铁线路
- 真实行政区划
- 真实交通规律（早晚高峰、工作日/周末差异）
